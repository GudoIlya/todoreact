1. Возможность запускать игру
- Вводится номер группы - является ключем к игре и кодовое слово - фамилия преподавателя и старосты группы
  три поля - код группы, фамилия препода, фамилия старосты

Эти же поля позволяют группе просмотреть свои предыдущие результаты, продолжить предыдущую игру, начать новую
Результаты игры сохраняются по ходу игры

Что из себя представляет игра.
1. Информация о группе - код группы, фамилия старосты
- включает информццию о кол-ве команд
-- команда - аватар, имя

2. Информация о теме игры
3. Информация о сложности игры
4. Информация о вопросах (json поле, в котором записываются id вопросов, id отвеченных вопросов id, информация о том, какой вопрос какой группе был назначен в ответы
5. Информация об очках каждой команды
6. Информация о команде - кол-во человек(можно не заполнять), аватар, название - команды привязаны к группам - информация о группе включает в себя


2. После первого пункта переходим к выбору темы
- Пользователи видят список из предложенных активных тем - каждой теме соответствует определенное кол-во вопросов
- можно сделать 3 варианта - 12-24-36 вопросов - быстрая, обычная, долгая игра

Соответственно, таблицы
1. Группы (код группы, Фамилия Старосты, команды - json поле с массивом команд - ключ команды, имя команды, аватар)
2. Игры (id группы, id темы, список вопросов, список отвеченных вопросов, информация о очках команды по ключу команды)
3. Вопросы (id темы, вопрос)
4. Темы (темы)
